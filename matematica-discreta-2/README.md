# Matemática Discreta II

Este repositorio contiene la entrega del proyecto para la materia Matemática Discreta II, año 2019. El objetivo principal es la implementación de una estructura de datos para grafos y algoritmos de coloreo en lenguaje C. Éste debe ser capaz de capaz de leer grafos desde entrada estándar, representarlos en memoria, ejecutar algoritmos de coloración y reordenamiento de vértices
