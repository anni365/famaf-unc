# Ingeniería del Software I

Este repositorio contiene el proyecto principal desarrollado para la asignatura **"Ingeniería del Software I"**.

El objetivo central de la materia fue la aplicación práctica de metodologías ágiles y el ciclo de vida del desarrollo de software, cubriendo desde la conceptualización y la captura de requisitos hasta la implementación de una aplicación web y las pruebas unitarias.

---

## Objetivos Clave de la Materia

Los temas centrales cubiertos en el proyecto fueron:

* **Análisis y Planificación:** Definición precisa del **Alcance del Proyecto** y el modelado de los **Casos de Uso** para delimitar el sistema. 
* **Diseño de Arquitectura:** Creación de **Diagramas de Flujo de Datos** y **Diagramas de Clases**.
* **Metodología SCRUM:** Aplicación práctica de la metodología SCRUM a través de la ejecución de **Sprints** definidos. Participación activa en las **Dailys** y rotación en el rol de **Scrum Master** para gestionar y coordinar el progreso del equipo. 
* **Pruebas:** Implementación de pruebas unitarias para garantizar la calidad del código.
