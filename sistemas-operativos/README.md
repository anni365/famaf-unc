# Sistemas Operativos - Proyectos Basados en xv6

Este repositorio contiene los proyectos y prácticas (Laboratorios) realizados para la asignatura **Sistemas Operativos**, basados en la versión de enseñanza del sistema operativo **xv6**.

## Acerca de xv6

**xv6** es un sistema operativo educativo, desarrollado en el MIT, que sirve para estudiar cómo funcionan los sistemas operativos estilo Unix. Está escrito en C y pensado para una arquitectura de 32 bits.

## Estructura del Proyecto

El repositorio está organizado por laboratorios, donde cada carpeta corresponde a una entrega o trabajo práctico específico.
